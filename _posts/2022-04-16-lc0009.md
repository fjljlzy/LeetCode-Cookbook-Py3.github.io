#### 代码1：
```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0:
            return False
        origin = x
        reverse = 0
        while x:
            reverse = reverse * 10 + x % 10
            x //= 10
        return reverse == origin
```

#### 代码2：
```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        return str(x) == str(x)[::-1]
```
