$$$$
标准模拟题，按照需求一步一步写代码。如何写的简约优雅是关键。

#### 代码
```python
class Solution:
    def myAtoi(self, s: str) -> int:
        # 去掉左边的前导空格

        s = s.lstrip()
        if not s:
            return 0
        # 处理加号减号

        if s[0] == '-':
            sign = -1
        else:
            sign = 1
        if s[0] == '+' or s[0] == '-':
            s = s[1:]
        # 将字符型数字转换为整数
        
        res = 0
        for char in s:
            if char.isdigit():
                res = res * 10 + ord(char) - ord('0')
            else:
                break
        res *= sign
        return max(-2**31, min(res, 2 ** 31 - 1))
```
